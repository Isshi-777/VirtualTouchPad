# VirtualTouchPad

仮想タッチパッド  
※IPointerDownHandlerやIPointerUpHandlerを実装する方ではないやり方

## 使用例
範囲指定をして下記の操作を検知し、セットされたイベントを実行する機構  
・タッチ（押した瞬間）  
・長押し  
・クリック（タップ）  
・ドラッグ  
・フリック  
    
※CanvasGroupのBlockRaycastなどは一切影響なし。
